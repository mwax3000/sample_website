Important HTML tags

This page is designed as a quick reference for some of the common tags covered in Learn Enough HTML to be Dangerous.In the process of making it, we'll learn how make HTML tables via table and related tags.

The tables below don't include all HTML tags, but they do list many of the most important ones.

Block Elements

Tag Name Purpose
  1. h1–h6
 headings include a heading(levels 1–6)
  1. p
 paragraph include a paragraph of text
  1. table
 table include a table
  1. tr
 table row include a row of data
  1. th
 table header make a table header
  1. td
 table data include a table data cell

Inline Elements

Tag Name Purpose Example Result
  • em
 emphasized make emphasized text <em>technical sophistication</em> technical sophistication
  • strong
 strong make strong text <strong>at least a billion people</strong> at least a billion people
  • a
 anchor make a hyper link <a href="https://learnenough.com/">Learn Enough</a> Learn Enough
  • img
 image include an image <img src="https://bit.ly/1MZAFuQ" alt="Michael Hartl">

Document Tags

Tag Name Purpose
  • html
 html make html document
  • head
 head make hidden header
  • body
 body make content section
  • title
 title make page title
  • meta
 meta specify meta settings like charset
  • span
 span hook to inline-elements
  • div
 division make a block of content
  • blockquote
 blockquote indent text
  • ol
 ordered list make numbered list
  • ul
 unordered list make bulleted list
